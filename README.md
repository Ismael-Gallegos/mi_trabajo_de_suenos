# Mi trabajo de sueños

Este repositorio contiene mi página web del Reto 2 del módulo de Introducción al Desarrollo Web de UVEG.  
La página fue creada usando HTML y CSS.

## Contenido del proyecto

- `index.html`: estructura básica de la página.
- `styles/styles.css`: estilos aplicados.
- `img/trabajo.jpg`: imagen alusiva a ciberseguridad.

## Descripción

La página muestra:
- Un título llamado “Mi trabajo de sueños”.
- Un párrafo donde describo mi trabajo ideal.
- Una lista con tres razones por las que me gustaría ese trabajo.
- Una tabla con pasos o requisitos.
- Una imagen relacionada con el tema.
- Uso de etiquetas `div`, `span`, `ul`, `li`, `table`, `img`.
- Estilos básicos como color, background, margin, padding, width, height y selectores.

## Cómo visualizarlo

Solo abre el archivo `index.html` en tu navegador.
